# 👨‍💻 Portfolio Arturo Morales Celis

Portfolio personal moderno y responsivo construido con React, Vite, Tailwind CSS y Framer Motion.

## 🚀 Características

- ✨ **Diseño Moderno**: Interfaz limpia y profesional con gradientes vibrantes
- 🎨 **Animaciones Fluidas**: Uso extensivo de Framer Motion para transiciones suaves
- 📱 **Responsive**: Totalmente adaptable a todos los dispositivos
- 🎯 **Performance Optimizado**: Carga rápida y eficiente
- 🔄 **Scroll Reveal**: Animaciones que aparecen al hacer scroll
- ⬆️ **Scroll to Top**: Botón flotante para volver al inicio
- 🎭 **Loading Screen**: Pantalla de carga animada
- 💫 **Header Sticky**: Navegación que cambia al hacer scroll

## 🛠️ Tecnologías

- **React 18** - Librería de UI
- **Vite** - Build tool ultrarrápido
- **Tailwind CSS** - Framework de CSS utilitario
- **Framer Motion** - Librería de animaciones
- **Iconify** - Iconos vectoriales
- **Swiper** - Carruseles (legacy, reemplazado por grid)

## 📦 Instalación

```bash
# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Build para producción
npm run build

# Vista previa del build
npm run preview
```

## 🎨 Secciones

1. **Hero/Introducción** - Presentación personal con animaciones
2. **Skills** - Tecnologías y herramientas con iconos animados
3. **Proyectos en Producción** - Grid de proyectos profesionales
4. **Proyectos Personales** - Desarrollos y prácticas personales
5. **Footer** - Enlaces de contacto y redes sociales

## 🎭 Componentes Principales

- `Header` - Navegación sticky con efecto de scroll
- `Introduce` - Hero section con animaciones de entrada
- `SkillsComponent` - Grid de skills con hover effects
- `ProjectCard` - Cards modernas para proyectos
- `ScrollToTop` - Botón flotante para volver arriba
- `LoadingScreen` - Pantalla de carga inicial

## 📝 Customización

Los colores principales utilizan gradientes de:
- Purple: `#9333ea` - `#7e22ce`
- Pink: `#ec4899` - `#db2777`
- Blue: `#2563eb`

Puedes modificar estos en `tailwind.config.js` y `index.css`.

## 🌐 Deploy

Este proyecto está listo para ser deployado en:
- GitHub Pages
- Vercel
- Netlify
- Cualquier hosting estático

```bash
npm run build
# Los archivos estarán en /dist
```
